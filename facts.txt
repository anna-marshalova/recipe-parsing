Рецепт чак-чака : 
Отделить желтки двух яиц . 
	Step
	{
		Action = отделить
		Object = желток
	}
Взбить 3 куриных яйца и 2 желтка до крутой пены . 
	Ingredient
	{
		Product = куриное яйцо
		Quantity = 3
	}
	Step
	{
		Action = взбить
		Object = 3 куриных яйца
	}
	Ingredient
	{
		Product = до
		Quantity = 2
		Measurement = желток
	}
В 300 г муки добавить взбитые яйца , 1/2 ч . ложки соли , 1 ч . ложка соды и 1 ч . ложка сока лимона . 
	Ingredient
	{
		Product = мука
		Quantity = 300
		Measurement = г
	}
	Step
	{
		Action = добавить
		Object = 300 г муки
	}
	Step
	{
		Action = добавить
		Object = взбитое яйцо
	}
	Step
	{
		Action = солить
		Object = ложок
	}
Тщательно вымесить тесто . 
	Step
	{
		Action = вымесить
		Object = тесто
	}
Оставить тесто на 15 минут . 
	Step
	{
		Action = оставить
		Time = 15 минут
	}
Разрезать тесто на несколько частей , сформировать тонкие жгутики . 
	Step
	{
		Action = разрезать
		Object = тесто
	}
	Step
	{
		Action = сформировать
		Object = тонкий жгутик
	}
Нарезать тесто на маленькие шарики . 
	Step
	{
		Action = нарезать
		Object = тесто
	}
Обвалять все шарики в муке . 
	Step
	{
		Action = обвалять
		Object = весь шарик
	}
Перед тем как обжарить шарики , сложите шарики в дуршлаг и стряхните с них лишнюю муку . 
	Step
	{
		Action = обжарить
		Object = шарик
	}
	Step
	{
		Action = сложить
		Object = шарик
	}
	Step
	{
		Action = стряхнуть
	}
Разогреть растительное масло и обжарить шарики несколькими порциями до золотистого цвета , 1-2 минуты . 
	Step
	{
		Action = разогреть
		Object = растительное масло
		Time = 1-2 минуты
	}
Растопить 150 мл мёда в сковороде или небольшом сотейнике . 
	Ingredient
	{
		Product = мед
		Quantity = 150
		Measurement = мл
	}
	Step
	{
		Action = растопить
		Object = 150 мл мёда
	}
Затем добавить 100 г сахарного песка и растопить на среднем огне , продолжать варить еще в течение 3 минут . 
	Ingredient
	{
		Product = сахарный песок
		Quantity = 100
		Measurement = г
	}
	Step
	{
		Action = добавить
		Object = 100 г сахарного песка
		Time = 3 минут
	}
Вылить 2/3 медового сиропа в шарики . 
	Ingredient
	{
		Product = медовый сироп
		Quantity = 2/3
	}
	Step
	{
		Action = вылить
		Object = 2/3 медового сиропа
	}
Получившуюся массу хорошо перемешать . 
	Step
	{
		Action = перемешать
	}
Мокрыми руками выложить шарики на тарелку и сформировать в форме пирамиды . 
	Step
	{
		Action = выложить
		Object = шарик
	}
	Step
	{
		Action = сформировать
	}
Сверху полить оставшимся сиропом . 
	Step
	{
		Action = полить
	}
Поставить в холодное место на 1-2 часа . 
	Step
	{
		Action = поставить
		Time = 1-2 часа
	}
Чак-чак готов . 
Приятного аппетита !
